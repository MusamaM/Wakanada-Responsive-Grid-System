The fastest way to create responsive multi-device layouts

First add an div with a class wak-container. This way you will create a vertical column container that will hold all the boxes inside. Then add some divs inside with the class wak-box-xx.

				<div class="wak-container">
					 <div class="wak-box-6"></div> 
					 <div class="wak-box-6"></div>

Offset Boxes

You can move blocks up to 11 boxes to the right by using the class wak-box-offset-xx 

				<div class="wak-container">
					<div class="wak-box-3"></div> 
					<div class="wak-box-6 wak-box-offset-3"></div> 
			 	</div>

Fluid Container

You can also use the fluid container by using the class wak-container-fluid

				<div class="wak-container-fluid">
					 <div class="wak-box-6"></div> 
					 <div class="wak-box-6"></div>
